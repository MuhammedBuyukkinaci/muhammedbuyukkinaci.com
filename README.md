# Steps

1)[Install .Net Core 3.0 on Ubuntu 18.04](https://dotnet.microsoft.com/download/linux-package-manager/ubuntu18-04/sdk-current)

2)Open up a new terminal check if .Net Core is istalled or not by the commmand below.

`
dotnet --version
`

3)Run commands below to create a new .net project on your Desktop

`
cd Desktop
`

`
mkdir mywebsite
`

`
cd mywebsite
`

`
dotnet new mvc
`

4)Run the craeted by proect via the command below:

```
dotnet run
```

5)Copy localhost:5000 and paste it on your favorite browser. You should see that application is running on your browser.
